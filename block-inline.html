block-level elements

block level elements are those that create a block formating context.
they are typically start on a new line and take up the fil width of their parent container

this is a block-level divelements

a common property of block-level elements is that they stack vertically.
each block level element starts on a new line below the previous one

inline level element

inline-level element,on the other hand, do not start on a new line and only take up as much width as necessary.they flow within the context.side by side

an example of an inline -level element is the strong element. it make text bold without starting a new line

inline element can appear withinblock-level elements,such as paragraphs and headings. this allows for inline staying and markup within larger content blocks.

mixed use

html documents often contain a mix of block-level and inline-level elements to structure content effectively for example,you might use headings(h1,h2) as block-level elements to define section of a page and then us inline-level elements like strongandemphasizedtext within those sections

the blockquote element is a block-level elements used for long quotations.it can contain both block-level and inline-level content
